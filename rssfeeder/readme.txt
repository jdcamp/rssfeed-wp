RSS Autoblog is an autoblogger plugin for generating content for your site. Uses rss feeds to generate and post the articles.

== Installation ==
Use WordPress’ Add New Plugin feature, searching “RSS Autoblog”, or download the archive and:

Unzip the archive on your computer
Upload rss-autoblog directory to the /wp-content/plugins/ directory
Activate the plugin through the ‘Plugins’ menu in WordPress
